01 — MUAMMO

Nima uchun kerak?

📧
Email jo'natish
Email server bilan ulanish, xabar yuborish — bu to'xtab turadi. Foydalanuvchi 5-10 sekund kutmasin.
~5–10 sekund
📁
CSV / PDF eksport
Minglab qatorni faylga yozish vaqt oladi. Background da bajarilib, foydalanuvchi xabar olsin.
~10–30 sekund
🖼️
Rasm qayta ishlash
Rasm kompressiya, resize, thumbnail — CPU ni band qiladi. Celery workerga topshiring.
~2–15 sekund
💥 Sinxron (yomon)
Foydalanuvchi so'rov yuboradi → Server ishni bajaradi → Foydalanuvchi kutadi → Javob keladi. UI qotib qoladi.
✅ Asinxron (yaxshi)
Foydalanuvchi so'rov yuboradi → task_id qaytariladi → Celery orqada ishlaydi → Tayyor bo'lgach xabar beradi.
02 — TEXNOLOGIYALAR

Celery va Redis

Celery
Task Queue · Worker
Python uchun ochiq manba asinxron vazifalar kutubxonasi. Og'ir ishlarni orqa fonda bajaradi, ilovangiz tez javob beradi.
pip install celery
Redis
Message Broker · Cache
Tezkor xotira ma'lumotlar bazasi. Celery uchun broker — vazifalarni qabul qilib, workerlarga uzatadi. RAM da ishlaydi, juda tez.
pip install redis
TAQQOSLASH JADVALI
Vazifasi Tasklarni orqa fonda bajaradi Broker — xabar uzatuvchi vosita
Roli ⚙️ Worker (ishchi) 📬 Queue (navbat)
Ishlab turgan joyda RAM + CPU RAM (in-memory)

01
celery.py fayl yaratish
config/ papkasida celery.py fayl yarating. Celery instansiyasini yarating va Django settings bilan bog'lang.
config/celery.py → Celery('config') yarating
02
__init__.py ga import qo'shish
config/__init__.py da celery_app ni import qiling — Django ishga tushganda Celery ham avtomatik yuklansin.
from .celery import app as celery_app
03
settings.py ni sozlash
CELERY_BROKER_URL va CELERY_RESULT_BACKEND ni Redis bilan sozlang. Timezone ham belgilang.
CELERY_BROKER_URL = 'redis://localhost:6379/0'
04
tasks.py da task yozish
Har bir app ichida tasks.py fayl yarating. @shared_task dekoratoru bilan funksiyalarni belgilang.
@shared_task   def send_email(...): ...
05
Celery Worker ishga tushirish
Alohida terminalda Celery workerni yoqing. Windows uchun --pool=solo parametri shart.
celery -A config worker --pool=solo -l INFO
04 — KOD

Amaliy kodlar

import os
from celery import Celery

# Django settings modulini ko'rsatamiz
os.environ.setdefault(
    'DJANGO_SETTINGS_MODULE', 'config.settings'
)

# Celery instansiyasi yaratamiz
app = Celery('config')

# Settings dan CELERY_ boshlanadigan o'zgaruvchilarni o'qiymiz
app.config_from_object('django.conf:settings', namespace='CELERY')

# Barcha app lardagi tasks.py fayllarni avtomatik topadi
app.autodiscover_tasks()
# config/__init__.py
# Django ishga tushganda Celery ham yuklansin

from .celery import app as celery_app

__all__ = ('celery_app',)
# settings.py — Celery sozlamalari

# Redis broker sifatida
CELERY_BROKER_URL = 'redis://localhost:6379/0'

# Natijalar Redis da saqlansin
CELERY_RESULT_BACKEND = 'redis://localhost:6379/0'

# Task boshlanganini kuzatamiz
CELERY_TASK_TRACK_STARTED = True

# Maksimal vaqt: 30 daqiqa
CELERY_TASK_TIME_LIMIT = 1800

# Toshkent vaqt zonasi
CELERY_TIMEZONE = 'Asia/Tashkent'
# myapp/tasks.py
from celery import shared_task
import time

# Oddiy task
@shared_task
def add(x, y):
    return x + y

# Email jo'natish simulatsiyasi
@shared_task
def send_email_task(email, subject):
    time.sleep(5)   # email yuborish vaqti
    return f"✅ Email {email} ga yuborildi: {subject}"

# CSV eksport
@shared_task
def export_csv_task(data):
    time.sleep(10)  # og'ir ish
    return "✅ CSV fayl tayyor!"
# views.py
from rest_framework.response import Response
from celery.result import AsyncResult
from config.celery import app
from .tasks import add

# Task ishga tushirish → task_id qaytaradi
@action(detail=False, methods=['get'])
def start_task(self, request):
    a = add.delay(1, 2)  # background ga yuborish!
    return Response({"task_id": a.id})

# Natijani tekshirish
@action(detail=False, methods=['post'])
def check_task(self, request):
    task_id = request.data['id']
    result = AsyncResult(task_id, app=app)

    if result.ready():  # Tayormi?
        return Response({"result": result.result})
    return Response({"status": "processing..."})
